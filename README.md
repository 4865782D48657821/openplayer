# OpenPlayer

An open source, hackable audio player platform.

## Vision

OpenPlayer is a portable audio device designed for focused listening and experimentation.

The goal is to create a stable music playback core that can be modified, extended, and explored without breaking the fundamental listening experience.

## Core values

- Open source
- Hackable
- Repairable
- Stable core
- Developer friendly

## Project goals

Primary goal:

Build a personal audio platform for experimenting with interaction, playback concepts, and music workflows.

Secondary goal:

Create a reproducible open hardware/software platform.

## Architecture philosophy

Stable core + flexible edges.

Audio playback must always work.
Experiments must never break core functionality.
