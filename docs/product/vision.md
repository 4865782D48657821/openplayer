# Product Vision

OpenPlayer is a portable open audio device designed for:

Focused listening.
Experimentation.
Long-term ownership.

It is not intended to compete with smartphones.

It is intended as:

A listening tool.
A development platform.
A personal audio lab.

## Core idea

Stable playback core.
Hackable outer layers.

## Non-goals

Not a streaming device.
Not an app ecosystem.
Not a general purpose computer.

## Experience goals

Calm interaction.
Physical controls.
Reliable behavior.

## Technical goals

Linux based.
Service architecture.
Event driven.
Modular design.
