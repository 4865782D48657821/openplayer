# Service Architecture

## playerd

Responsibilities:

Command routing.
State management.
MPD integration.
Event broadcasting.

## inputd

Responsibilities:

GPIO reading.
Debounce.
Input events.

## uid

Responsibilities:

Rendering.
Navigation.
User interaction.

## sysd

Responsibilities:

Battery.
Shutdown.
System health.

## mpd

Responsibilities:

Audio playback.
Queue.
Track state.
